= CoreCrypto
:hide-uri-scheme:

This library is a simplified abstraction over the MLS and Proteus cryptographic protocols for end-to-end encrypted commmunication.
In addition, it provides end-to-end identity (E2EI) support.

It is available for a number of platforms and languages. For detailed documentation for a given platform/language pair, use the index to the left. Alternatively, consult the language references for a given version directly at https://wireapp.github.io/core-crypto.
