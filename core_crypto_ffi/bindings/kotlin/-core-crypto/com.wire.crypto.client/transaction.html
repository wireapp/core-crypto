transaction

suspend fun <R> CoreCryptoCentral.transaction(block: suspend (context: CoreCryptoContext) -> R): R?

Starts a transaction in Core Crypto. If the callback succeeds, it will be committed, otherwise, every operation performed with the context will be discarded.

Return

the return of the function passed as parameter

Parameters

block

the function to be executed within the transaction context. A CoreCryptoContext will be given as parameter to this function
